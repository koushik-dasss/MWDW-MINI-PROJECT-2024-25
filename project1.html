Playback Speed Controller for Spotify

This is a full stack MERN project where I implemented a custom playback speed controller for Spotify's music and podcasts. It allows users to adjust the playback speed to suit their needs, improving the listening experience.

Features:

  • Login with Spotify account (OAuth)
  • Change playback speeds: 0.5x, 1x, 1.5x, and 2x
  • Real-time control of Spotify playback

Technologies Used:

MongoDB, Express.js, React.js, Node.js, Spotify Web API
